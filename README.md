# funny-friends-site

A small Next.js site for sharing funny friend moments: short posts, quotes, and inside jokes.

## Tech

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run start` — run production server
- `npm run lint` — lint

## Deploy

This can be deployed to Vercel or any Node host that supports Next.js.
